# cost-of-comparison
A repository for comparing the cost of living in different cities. 

Input: City A
Input: City B
Input: Occupation 

Output: Net Income
Based on:
- Housing cost https://www.zillow.com/research/data/
- Food: Compare 10 different items at once (webscrape): https://www.bls.gov/regions/mid-atlantic/data/averageretailfoodandenergyprices_usandwest_table.htm
- Energy: Compare multiple (webscrape): https://www.bls.gov/regions/mid-atlantic/data/averageretailfoodandenergyprices_usandwest_table.htm 
- Health Insurance: 2025 Data (Fixed): https://worldpopulationreview.com/state-rankings/health-care-costs-by-state
- Taxes: https://api-ninjas.com/api/incometaxcalculator
- Income: https://www.bls.gov/developers/api_python.htm#python1
